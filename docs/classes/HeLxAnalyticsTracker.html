  • trackEvent(event: TrackingEvent): Promise<TrackingResponse>
    • Defined in Analytics.ts:146

    Tracks a specific event (e.g. visiting a page, clicking a button, etc.) by sending it to the analytics platform.

    TrackingEvent is broad and should contain data required by all trackers (which has great overlap).

    This method should choose/transform the data specifically relevant to the implementation's tracking platform and relay it accordingly.

    Note: implementation should call @trackingEvent
